# Audio Spectrum Visualizer

A real-time audio spectrum visualizer built with TypeScript, HTML5 Canvas, and the Web Audio API. 

## Features
* **Time Domain (Waveform):** Displays the raw audio wave in real-time.
* **Frequency Domain (FFT):** Displays frequency bars mapped to the audible spectrum, featuring a dynamic rainbow color mapping.
* **Peak Detection:** Calculates and displays the dominant frequency (Hz) currently being detected by the microphone.

## Prerequisites
Because this project uses TypeScript and Vite as a build tool, you will need to have [Node.js](https://nodejs.org/) installed on your computer.

## How to Run Locally

1. **Install Dependencies:**
   Open your terminal in the project folder and install Vite by running:
   `npm install`

2. **Start the Development Server:**
   Run the following command to start the local Vite server. This will automatically compile the TypeScript and open the app in your default web browser:
   `npm start`

3. **Stopping the Server:**
   When you are finished, closing the browser tab is not enough. You must return to your terminal and press `Ctrl + C` to kill the local server.
